## 🎯 О проекте

**SurveyApp** — бесплатное веб-приложение с открытым исходным кодом для создания онлайн-опросов и тестов. Система позволяет за несколько минут сконструировать анкету любой сложности, поделиться уникальной ссылкой с респондентами и получить наглядные результаты в виде интерактивных диаграмм.

Проект разработан в рамках выпускной квалификационной работы на тему **«Веб-приложение для создания и анализа онлайн-опросов»**.

## ✨ Возможности

### 📝 Конструктор опросов
- **5 типов вопросов:** текстовый ответ, один вариант, несколько вариантов, шкала (1-5), да/нет
- Настройка обязательности каждого вопроса
- Режим теста с указанием правильных ответов
- Управление статусами: черновик → активный → закрыт

### 🔗 Распространение
- Уникальная ссылка для каждого опроса
- Прохождение без регистрации
- Копирование ссылки в один клик

### 📊 Аналитика
- Интерактивные диаграммы (круговые, столбчатые)
- Подсчёт процентов, средних значений
- Просмотр текстовых ответов
- Экспорт результатов в CSV

### 🎓 Режим теста
- Автоматический подсчёт баллов
- Показ результата респонденту после прохождения
- Отметка правильных и неправильных ответов

### 👤 Управление аккаунтом
- Регистрация и авторизация (JWT)
- Безопасное хранение паролей (bcrypt)
- Личный кабинет со списком опросов

## 🚀 Быстрый старт

### Предварительные требования

- [Python 3.11+](https://python.org/downloads/)
- [MongoDB 7.0+](https://www.mongodb.com/try/download/community)
- [Git](https://git-scm.com/download/)

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Sirellia/SurveyApp.git
cd SurveyApp

# 2. Создайте виртуальное окружение
python -m venv venv

# Windows
venv\Scripts\activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Создайте файл .env
echo SECRET_KEY=your-secret-key-change-this > .env
echo MONGO_URI=mongodb://localhost:27017/survey_app >> .env

# 5. Запустите приложение
python app.py
